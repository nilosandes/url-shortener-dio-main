# url-shortener-dio
Projeto base feito pela especialista da DIO Alexia Pereira. Projeto desenvolvido no Bootcamp Spread Fullstack Developer na plataforma da Digital Innovation One.

Projeto criado junto da DIO para construção de um Encurtador de URL.

## Rodando o projeto

- `npm install`
- `npm run build`
- `npm run dev`

Algumas informações sobre os vídeos:

- Editor de Texto usado: VSCode
  - Color Theme do editor: Material UI e Bluloco Dark
- Programa para verificar as requests: Insomnia
